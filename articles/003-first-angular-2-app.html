Hello world Angular 2 app

Mario Romano   25. May, 2016
Let's see how to write our first app with Angular 2 (rc.1).

How to install it

Let’s start cloning a quick start project for Angular 2: https://github.com/magemello/angular2-sample-project Type in the terminal: 
                                
                                    git clone https://github.com/angular/quickstart
                                    cd quickstart
                                    npm install
                                    npm start
                                    npm run test
